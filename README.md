# BraiNO

## Vision

The projects vision is to offer a platform for collecting your personal information along your lifetime within a searchable structured format.

## Design goals
- Open Source
- Privacy first
- Security by design
- Decentral
- Easy data input
- Platform
- Extendable
- Plugins
